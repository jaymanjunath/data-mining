# data-mining
Mining Tweets 

Setup github repo - 
git clone git@github.com:jaymanjunath/data-mining.git

Set up origin - 
git remote set-url origin git@github.com:jaymanjunath/data-mining.git

Set up python env - 
cd data-mining 
pyenv virtual env <name>
pyenv local <name>
pip install -r requirements.txt
